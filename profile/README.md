# HenosisLab

> 万物归一 · 智能体实验室

HenosisLab 是一个专注于 AI 智能体、本地模型运行时和开发者工具的研究与开发组织。

---

## 产品矩阵

| 项目 | 描述 | 状态 |
|------|------|------|
| [AgentKeeper](https://github.com/henosislab/agentkeeper) | 智能体框架，让 AI 自主执行任务 | 🔄 开发中 |
| [AgentMindset](https://github.com/henosislab/agentmindset) | 任务编排与调度系统，定义智能体的思维方式 | 🧠 设计阶段 |
| [Aithna](https://github.com/henosislab/aithna) | 本地模型运行时，轻量级推理引擎 | 🔄 开发中 |
| [Edger Editor](https://github.com/henosislab/edger-editor) | AI 增强的文本编辑器 | 📝 规划中 |
| [Weft Terminal](https://github.com/henosislab/weft-terminal) | AI 原生 SSH 客户端 | 🔄 开发中 |
| [CCBL License](https://github.com/henosislab/ccbl-license) | 自定义开源协议框架 | 📄 稳定 |

---

## 核心理念

- **本地优先**：数据主权归属用户
- **模块化**：每个组件可独立使用
- **开放协议**：CCBL 协议保障生态健康
